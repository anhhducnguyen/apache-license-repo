# Simple Interest Calculator

## Description
This project is a simple interest calculator that computes interest based on principal amount, interest rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

Where:
- Principal = Initial amount of money
- Rate = Annual interest rate (%)
- Time = Time period in years

## Example
If:
Principal = 1000
Rate = 5%
Time = 2 years

Simple Interest = (1000 × 5 × 2) / 100 = 100

## Usage
Enter the principal amount, rate of interest, and time period to calculate the simple interest.
